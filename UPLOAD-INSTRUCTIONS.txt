WORLD CUP DASHBOARD - GITHUB UPLOAD PACKAGE
===========================================

This folder contains EVERYTHING needed for GitHub Pages + auto-updates.
Upload the CONTENTS of this folder to your repo root (not the github-upload folder itself).

STRUCTURE (do not rename folders):
  .github/workflows/update-world-cup-dashboard.yml
  .gitignore
  FIFA Draw(Draw Results).csv
  assets/cisco-brand.css
  World Cup/  (4 source files)
  docs/       (live site - index.html, world-cup-data.js, assets/)

UPLOAD STEPS:
1. Create a new repo at https://github.com/new (e.g. world-cup-dashboard)
2. On the repo page, click "Add file" -> "Upload files"
3. Drag ALL files and folders from THIS folder into the browser
   (include .github - you may need to select it from file picker)
4. Commit to main branch

ENABLE PAGES:
  Settings -> Pages -> Deploy from branch -> main -> /docs -> Save
  Live URL: https://YOUR-USERNAME.github.io/REPO-NAME/

ENABLE AUTO-UPDATE:
  Settings -> Actions -> General -> Workflow permissions -> Read and write -> Save
  Actions tab -> "Update World Cup Dashboard" -> Run workflow (test once)

After setup, scores refresh automatically every ~15 minutes.
